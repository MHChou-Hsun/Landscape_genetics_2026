# Landscape Genetics in R

## Overview

This repository contains R scripts for conducting population-based landscape genetic analyses, including:

- Pairwise environmental distance calculation
- Pairwise resistance distance calculation
- Mantel tests
- MLPE (Maximum Likelihood Population Effects) modeling

These scripts were developed for analyzing genetic differentiation in relation to geographic (isolation by distance, IBD), environmental (isolation by environment, IBE), and landscape resistance (isolation by resistance, IBR) factors.

## List of scripts and what they do

- `env_dist_calc.R`  
  Calculates pairwise environmental distances among populations from .asc raster files. 

- `resis_dist_calc.R`  
  Generates resistance distance matrices using landscape resistance surfaces. In this script, pairwise resistance distance among population is calculated using [ResistanceGA](https://github.com/wpeterman/resistancega).

- `mantel.rtest.R`  
  Performs Mantel tests to assess correlations between genetic and explanatory distances using the mantel.rtest function in [ade4](https://www.rdocumentation.org/packages/ade4/versions/1.7-23). Pairwise geographic distance among populations is also calculated using [geodist](https://github.com/hypertidy/geodist) in this script.

- `MLPE_landscpae_gen.R`  
  Fits MLPE models to evaluate IBD, IBR, and IBE hypotheses using [corMLPE](https://github.com/nspope/corMLPE). This script also calculates pairwise geographic distance among populations based on GPS coordinates.

## Example files

`MLPE_landscpae_gen.R` and `mantel.rtest.R` use distance matrices generated by `env_dist_calc.R` and `resis_dist_calc.R`. The example pairwise [Fst input](./examples/Fst.csv), [GPS coordinates](./examples/GPS.csv), [climate variables](./examples/clim_data/*.asc), and a [raster file](./examples/resis_surface/alboobliquatus.asc) are provided for testing the scripts. The climate variable files were downloaded from [WorldClim](https://www.worldclim.org/); the [raster input file](./examples/resis_surface/alboobliquatus.asc) was generated by species distribution modeling with [MaxEnt](https://biodiversityinformatics.amnh.org/open_source/maxent/), representing habitat suitability. The expected outputs are also provided.
